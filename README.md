# MoodFood 
This web app suggests music and food based on your current mood.

## Features
- Mood-based UI
- Dark mode toggle
- Spotify playlist embed

## Tech Stack
- HTML, CSS, JS
- Spotify Embed API

## How to Use
1. Select your mood
2. Enjoy matching music and food
3. Toggle dark mode for fun!
